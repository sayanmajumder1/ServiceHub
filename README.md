### Prerequisites:
- PHP 8.0 or higher
- MySQL 8.0 or higher
- Web server (Apache/Nginx)
- Modern web browser

### Installation Steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sayanmajumder1/ServiceHub.git
   ```

2. **Set up the database:**
   - Import the SQL file from `sql/database.sql`
   - Configure database credentials in `config/db_config.php`

   - Update base URL in configuration files
   - Set proper file permissions for uploads directory

4. **Access the application:**
   - Navigate to the project directory via web browser
